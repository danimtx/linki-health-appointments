# PROCESO DE ADMISIÓN TECH F2 - 2026
## LINKI HEALTH
## MÓDULO DE PROGRAMACIÓN DE CITAS

### CONTEXTO PARA EL DESAFÍO
Estamos desarrollando un módulo para nuestra plataforma digital de salud que permite a los pacientes gestionar sus citas médicas de manera simple y eficiente.
Uno de nuestros desafíos es mejorar el proceso de reserva de citas para ofrecer una experiencia rápida, intuitiva y accesible desde cualquier dispositivo.
Buscamos personas capaces de diseñar soluciones centradas en el usuario, con una excelente experiencia digital y pensando en la escalabilidad del sistema.

### TECHNICAL CHALLENGE
**Diseña y desarrolla un Módulo Web de Programación de Citas**

**Tu solución debe permitir que un paciente:**
* Seleccione una especialidad médica.
* Elija un profesional disponible.
* Visualice fechas y horarios disponibles.
* Complete sus datos personales.
* Confirme exitosamente una cita médica.

### REQUISITOS MÍNIMOS
* **Especialidades:** El usuario debe poder elegir entre distintas especialidades médicas.
* **Profesionales:** Mostrar profesionales disponibles según la especialidad seleccionada.
* **Agenda Médica:** Visualizar horarios disponibles y evitar reservas duplicadas.
* **Confirmación:** Mostrar un resumen de la cita antes de finalizar la reserva.
* **Diseño Responsivo:** La experiencia debe funcionar correctamente en móvil y escritorio.

### BONUS
**Se valorará positivamente:**
* Calendario interactivo
* Integración con API simulada
* Validaciones avanzadas
* Modo oscuro
* Accesibilidad web
* Dashboard básico de citas
* Conexión a Base de Datos
